# Desenvolvimento Front-End

Repositório com projetos React em desenvolvimento.

## Projetos

### 🌤️ Weather API
**Versão:** 0.0.1  
**Descrição:** Aplicativo de previsão do tempo com busca por cidades do Brasil.

**Tecnologias:**
- React 19
- TypeScript
- Vite
- CSS Responsivo

**Como usar:**
```bash
cd Weather-api\ [DEV]/Weather_API

# Instalar dependências
npm install

# Executar em desenvolvimento
npm run dev

# Build para produção
npm build

# Verificar linting
npm run lint
```

**Variáveis de ambiente:**
Crie um arquivo `.env.local` na raiz do projeto:
```
VITE_API_KEY=sua_chave_api_aqui
```

**Recursos:**
- Busca de cidades por nome
- Exibição de temperatura máxima e mínima
- Interface responsiva
- Validação de cidades
